# CRM Dataset Builder

A portfolio project for generating realistic CRM (Customer Relationship Management) data.

## Overview

This tool generates synthetic CRM datasets that mimic real-world customer data patterns. Useful for:

- Testing CRM applications
- Demonstrating data analysis workflows
- Populating development databases
- Training and educational purposes

## Project Structure

```
crm-dataset-builder/
├── src/          # Source code
├── output/       # Generated datasets
├── examples/     # Example configurations and sample outputs
└── README.md
```

## Getting Started

```bash
python src/main.py
```
## Built With
- Python 3.x
- Claude Code (AI-assisted development)
- Faker library for realistic data generation
- pandas for data manipulation
